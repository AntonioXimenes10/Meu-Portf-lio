# Meu Portfólio

Este é o meu portfólio pessoal, desenvolvido como parte do projeto "Crie um Site de Portfólio com HTML, CSS e JavaScript" do Coursera.

## Sobre Mim

Olá! Meu nome é Antonio Pedroza Ximenes, desenvolvedor web com formação em Análise e Desenvolvimento de Sistemas e experiência em criação de soluções tecnológicas. Tenho expertise em coleta, organização e análise de dados, utilizando ferramentas como Microsoft Power BI, SQL e Python, o que me permite transformar informações complexas em insights valiosos.

## Meu Projeto

Este projeto consiste em um site pessoal que destaca minhas habilidades e experiências profissionais. O site foi desenvolvido utilizando HTML, CSS e um pouco de JavaScript para interatividade.

## Funcionalidades

* Página "Sobre Mim" com uma breve introdução.
* Página "Meu Projeto" descrevendo o projeto do portfólio.
* Links para meu perfil do LinkedIn e GitHub.
* Seção "Minha Abordagem" explicando o processo de desenvolvimento.

## Como Executar

1.  Clone este repositório para o seu computador.
2.  Abra o arquivo `index.html` em seu navegador.

## Minha Abordagem

Adotei uma abordagem minimalista, priorizando a clareza e a facilidade de uso. O design é limpo e intuitivo, permitindo que os visitantes encontrem facilmente as informações que procuram. A navegação é direta e organizada, com seções bem definidas para cada aspecto do meu perfil profissional.
